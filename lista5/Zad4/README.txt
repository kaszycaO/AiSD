ZADANIE CZWARTE
***************

Przykladowe wywolanie:



python3 main.py --short < data/input.txt



Program posiada nastepujace parametry:

--short - algorytm zachlanny
--rand - losowe bladzenie
--eulerK - drzewo rozpinajace + Kruskal
--eulerP - drzewo rozpinajace + Prim



DODATKOWO:
Dodatkowo w folderze data znajduje sie skrypt do generowania inputu (zgodnie z trescia zadania)
Przykladowe wywolanie


python3 data_gen.py 5 > input.txt



Zmieniajac parametr (w tym przypadku 5) mozna wygenerowac input o dowolnej wielkosci.


KOMENTARZ:

W folderze data znajduje sie krotki komentarz, w pliku Testy.pdf, na temat dzialania algorytmow
